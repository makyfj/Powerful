# Powerful

Powerful is the simplest and most intuitive workout experience.

## Front End

- NextJs
- Redux Toolkit
- Redux Toolkit Query
- Tailwind

## Back End

- Prisma
- Express
- TypeScript
- PostgreSQL
- JWT
- Cookie Parser
